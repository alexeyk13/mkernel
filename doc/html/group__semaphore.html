Detailed Description

semaphore is a sync object. It's used, for signalling sync condition

Semaphore is quite simple. semaphore_signal increments counter, any semaphore_wait*, decrements. If counter reached zero, thread will be putted in waiting state until next semaphore_signal.

Because semaphore_wait, semaphore_wait_ms, semaphore_wait_us can put current thread in waiting state, this functions can be called only from SYSTEM/USER context. Other functions, including semaphore_signal can be called from any context
