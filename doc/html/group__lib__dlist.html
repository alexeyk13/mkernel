Detailed Description

dual-linked list routines

dual-linked list is used mainly for system objects, however, can be used for user lists.

DLIST is just a header, added to custom structure as a first element. Using cast it's possible to use any structure as dual-linked lists of custom objects.

For memory-critical apllications it's possible to use dual-linked lists also as a containers for up to 2 bit flags (for 32 bit system)
